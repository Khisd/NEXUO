Pertanyaan Umum (FAQ)

Apakah ini aman? +
Ya. NEXUO menggunakan parameter manajemen risiko ketat (Stop Loss wajib). Kami tidak menjamin profit, tapi kami meminimalisir kerugian besar dengan sistem proteksi modal.
Berapa modal minimal? +
Tergantung produk. Namun secara umum, minimal $100 (akun Cent/Standard) sudah cukup untuk memulai agar manajemen risiko berjalan efektif.
Saya gaptek, bisa install sendiri? +
Tentu bisa. Setelah pembelian, Anda akan mendapatkan file PDF dan Video tutorial langkah-demi-langkah. Prosesnya mirip menginstall aplikasi biasa.
Broker apa yang support? +
Produk kami kompatibel dengan broker MT5 universal dan akun TradingView (Gratis/Berbayar).
